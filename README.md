> <h3> Getting start : </h3>
> <br/>

<div>

1. <b>Clone the project</b>

```bash
  $ git clone https://github.com/gabriel-durr/design-system-storybook.git && cd design-system-storybook
```

2. <b>Install dependencies</b>

```bash
  $ npm i
```

3. <b>Start the application</b>

```bash
  $ npm run dev
  $ npm run storybook
```
